# 🐶 布鲁伊父母支持 (Bluey Parenting Skill)

基于布鲁伊（Bluey）动画片的育儿支持 Coze 技能。

## 这是什么？

当父母在育儿中遇到棘手问题、感到疲惫或需要支持时，这个技能会优先从布鲁伊剧集中班迪特和奇莉的应对方式寻找参考；找不到对应场景时，补充心理学观点并注明出处。

## 使用方法

### 方式一：上传到 Coze（推荐）

1. 下载本仓库的 zip 文件（点击绿色 Code 按钮 → Download ZIP）
2. 登录 [Coze](https://www.coze.cn)
3. 在对话框中告诉 Agent "上传技能zip"，将下载的 zip 文件传上去
4. Agent 会自动解压并进入编辑模式
5. 根据需要修改 SKILL.md（比如添加自己喜欢的博主观点）
6. 告诉 Agent "发布" 即可

### 方式二：自定义修改

1. Fork 或下载本仓库
2. 编辑 `SKILL.md`，按需调整：
   - 添加参考博主观点
   - 修改配置项（孩子年龄/性别/数量）
   - 调整回复风格
3. 打包为 zip 后上传到 Coze

## 技能配置

技能支持以下配置项（首次使用时自动引导填写）：
- 孩子年龄
- 孩子性别
- 孩子数量

## 技术细节

- 纯 prompt 技能，无第三方 API 依赖
- 核心文件：`SKILL.md`（包含完整的技能定义、prompt 模板和参考来源）
- 适用于 Coze 平台
